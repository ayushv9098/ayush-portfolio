# Ayush Vishwakarma Portfolio - Agent Guide

## Tech Stack
- **Framework:** Next.js 16 (Turbopack)
- **Styling:** Tailwind CSS 4, Framer Motion, GSAP
- **Icons:** Lucide React, Custom SVG Icons
- **Deployment:** Vercel

## Visual Feedback (Agentation)
This project uses **Agentation** for visual feedback and agent sync.
- The `Agentation` toolbar is available in development mode.
- Use it to annotate UI elements and sync with Gemini CLI.
- Endpoint is set to `http://localhost:4747` for default Gemini CLI integration.

## Project Structure
- `app/`: Next.js App Router files.
- `components/sections/`: Individual portfolio sections (Hero, About, etc.).
- `components/layout/`: Shared layout components (Navbar, Footer, CustomCursor).
- `public/`: Static assets (images, profile pictures).
